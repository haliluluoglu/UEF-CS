Exercise 6: Watershed and Fourier transforms

  1. Compute the watershed transform of a binary image containing two overlapping circles.

    1. Compute the distance transform (bwdist) of the complement of the image using Euclidean, chessboard and cityblock distance metrics. Show the distance transforms with mat2gray and imshow. How does the distance metric affect on the distance transform?

    2. Choose one of the distance transforms, complement it and force pixels not belonging to the objects have value -Inf (Matlab command for the operation is of form D1(~bw)=-Inf;).

    3. Now the watershed transform of the distance transform matrix can be calculated (watershed). To visualize the watershed transform, use label2rgb and imshow commands.

  2. Calculate the Fourier transform (fft2) of an image of a building. The result is complex (containing complex numbers). Illustrate both the natural logarithm of the magnitude (log, abs) and the phase (angle) of the result. (You might want to adjust the display range to make the visualization better; check help for imshow for how it is done. Usually the 2D Fourier transform is shifted so that the zero-frequency component is in the centre. This can be done with fftshift.)

    Creating image filters in frequency domain.

  3. Create highpass, lowpass, bandpass and bandreject 2D FIR filters, and filter some grayscale image with them (you can use, for example, test pattern image). Useful Matlab commands: firpm, freqz, ftrans2, freqz2, imfilter.
