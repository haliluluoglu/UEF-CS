Machine Vision

Exercise 8

    Image compression.

  1. The JPEG image compression algorithm is based on Discrete Cosine Transform. Repeat the compression example in the Matlab help page (you can use any grayscale image). How can the percentage of discarded DCT coefficients be adjusted?

  2. Compress a color image (e.g., the mandrill) with wcompress. What compression ratio and bit-per-pixel ratio you get? Try also the step-by-step uncompression (an example can be found in Matlab help).

    Creating image filters in frequency domain.

  3. Create highpass, lowpass, bandpass and bandreject 2D FIR filters, and filter some grayscale image with them (you can use, for example, test pattern image). Useful Matlab commands: firpm, freqz, ftrans2, freqz2, imfilter.
