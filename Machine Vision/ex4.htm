Exercise 4: Morphological Image Processing

Matlab's help has a chapter about morphology - search for "Morphological Operations". (The same text can be found on MathWorks webpage.)

  1. Get an image with broken characters. Use imdilate to dilate the image. The maximum length of the breaks is known to be two pixels.

    Dilated broken characters image

    1. The thin lines connecting the center region to border pads have to be removed from a binary image of a wirebond mask. The other structures have to be preserved. This can be done by choosing a structuring element small enough to fit within the center square and thicker border leads, but too large to fit entirely within the wires. Erode the wires using a suitable structuring element. After that, restore the center square and thicker border leads to their original size using dilation with the same structuring element.

      Wirebond mask with the thin wires removed

    2. Perform the same operation using morphological opening (imopen) with the same structuring element.

  3. Consider an image showing a region of small circles enclosed by a region of larger circles. (This exercise is easier to solve if you first complement the original image using imcomplement.) The image has two textural regions: the one composed of large blobs and the other composed of smaller blobs. Find the boundary between the two regions based on their textural content. Use morphological operations (opening and closing) for removing small blobs and combining large blob area into one combined area as

    The boundary can be found with morphological gradient (= the difference between the dilation and the erosion).
