Exercise 2: Color images

  1. Read a grayscale image and a color image to Matlab into different variables (commands f = imread('5.1.09.tiff'); and imshow(f) are useful). Check the variable sizes (in Workspace window, or use the whos command) - how do the dimensions differ between a grayscale and a color image? What do the dimensions represent? (You can try the Data Cursor Tool from Matlab figure toolbar: click on the image - what do the numbers mean?) Show the three different channels of the color image (for example, imshow(f(:,:,1))). What do you think they represent?

  2. Convert the color image RGB values into HSV (hue, saturation, value) and YIQ (luminance, hue, saturation) color spaces (rgb2hsv, rgb2ntsc). Visualize the channels in each color space (with imshow). What other color spaces does Matlab support? (search Help for "color space conversion".)

  3. There are many ways for converting color images into grayscale. One of them is rgb2gray. How is the conversion done? What color space contains the grayscale image as one component?

If you want to use Matlab for digital image processing, commands such as imrotate, imresize, imadjust and imcontrast can be useful.
